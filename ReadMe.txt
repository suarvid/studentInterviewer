This project utilizies the Furhat Remote API.
Therefore, it requires that the furhat_remote_api python module is installed.
This easiest way to do this is to run the command
	"sudo pip install furhat-remote-api"
if pip is installed and used as a package manager, or
	"sudo conda install furhat-remote-api"
if conda is installed and used as a package manager.
See the furhat documentation over the api at https://docs.furhat.io/remote-api/.
